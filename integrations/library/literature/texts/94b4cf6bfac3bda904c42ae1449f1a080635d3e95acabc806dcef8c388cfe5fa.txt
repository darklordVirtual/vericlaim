Lexical threats are, in a precise sense, rule-complete: a sufficiently careful pattern library resolves them with high precision. Semantic / intent-dependent threats. Here the danger does not inhere in any token. The malicious and the benign action share the same surface form; the verdict flips only on the destination, content, scope, or intent behind that form. The same curl is telemetry to a monitoring endpoint or exfiltration to an attacker; the same kubectl get secret is a debugging read or credential theft; the same database access is a routine SELECT or a bulk COPY ...
