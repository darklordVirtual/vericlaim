Now we present our vision on how to use LLM APIs within a budget. As shown in Figure 1 (b), we discuss three cost-reduction strategies, namely, prompt adaptation, LLM approximation, and LLM cascade. Figure 2: Illustrations of cost-saving strategies. (a) Prompt selection uses a subset of in-context examples as the prompt to reduce the size of the prompt. (b) Query concatenation aggregates multiple queries to share prompts. (c) Completion cache stores and reuses an LLM API’s response when a similar query is asked. (d) Model fine-tuning uses expensive LLMs’ responses to fine-tune cheap LLMs. (e) LLM cascade employs different LLM APIs for different queries.
