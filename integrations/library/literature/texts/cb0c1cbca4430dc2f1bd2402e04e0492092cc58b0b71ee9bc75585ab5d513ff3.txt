5.7.2.1 prov:label

The attribute prov:label ◊ provides a human-readable representation of an instance of a PROV-DM type or relation. The value associated with the attribute prov:label MUST be a string. Example 48 ◊

The following entity is provided with a label attribute. entity(ex:e1, [ prov:label="This is a human-readable label" ])
