In two-player zero sum games the value functions are assumed to be bounded within the interval. This choice allows us to
combine value estimates with probabilities using the pUCT rule (Eqn 2). However, since in many environments the value is unbounded, it is necessary
to adjust the pUCT rule. A simple solution would be to use the maximum score
that can be observed in the environment to either re-scale the value or set the
pUCT constants appropriately [33]. However, both solutions are
game specific and require adding prior knowledge to the MuZero algorithm. To avoid this, MuZero computes normalized value estimates
by using the minimum-maximum values observed in the search tree up to that point.
