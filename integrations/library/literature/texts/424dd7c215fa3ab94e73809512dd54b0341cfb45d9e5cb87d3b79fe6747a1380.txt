Our experiment produced 100 sets of rank data, where each participant ranked the five conditions by believability. To translate this rank data into interval data for interpretable comparison, we used the ranks to calculate a TrueSkill rating (Herbrich et al., 2006) for each condition. TrueSkill is a generalization of the Elo chess rating system (Elo, 1967) for a multiplayer environment, and has been used by Xbox Live for player ranking based on competitive game performance. Given a set of ranked outcomes, TrueSkill outputs a mean rating value and standard deviation for each condition. Conditions with the same rating should roughly be a toss-up, with each winning half of the comparisons between the two conditions. Higher scores indicate conditions that beat lower-ranked conditions in the rankings. Separately, to investigate the statistical significance of these results, we applied the Kruskal-Wallis test (Kruskal and Wallis, 1952), a non-parametric alternative to the one-way ANOVA, to the raw rank data. We then performed the Dunn post-hoc test (Upton and Cook, 2006) to identify any pairwise differences between the conditions.
