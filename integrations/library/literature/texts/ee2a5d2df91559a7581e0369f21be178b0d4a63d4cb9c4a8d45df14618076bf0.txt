GEM provides an environment in which models can
be tested on a wide set of tasks and with different evaluation strategies that can evolve alongside the models. A related project that grew out of GEM (and was inspired by BIG-bench) is
NL-Augmenter (Dhole et al., 2021),
which crowd-sources transformations and augmentation of textual datasets. The Natural Instructions Expansion project444https://github.com/allenai/natural-instructions-expansion has been initiated in an effort to source a large collection of natural-language tasks and their natural-language definitions/instructions. This project is an expansion of NATURAL INSTRUCTIONS (Mishra et al., 2021), which contained a collection of 61 tasks. DynaBench (Kiela et al., 2021) is an open-source platform for dynamic dataset creation and model benchmarking that uses human-and-model-in-the-loop to identify and collect examples where models will fail but humans would not. SyntaxGym (Gauthier et al., 2020) is an online platform designed to make targeted evaluations following the norms of psycholinguistic experimental design.
