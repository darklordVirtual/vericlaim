ReAct yao2022react leverages chain-of-thought prompting wei2022chain and generates both reasoning traces and task-specific actions with LLMs. Reflexion shinn2023reflexion is built upon ReAct yao2022react with self-reflection to enhance reasoning. AutoGPT autogpt is a popular tool that automates NLP tasks by crafting a curriculum of multiple subgoals for completing a high-level goal while incorporating ReAct yao2022react ’s reasoning and acting loops. DERA nair2023dera frames a task as a dialogue between two GPT-4 openai2023gpt4 agents. Generative Agents park2023generative leverages ChatGPT chatgpt to simulate human behaviors by storing agents’ experiences as memories and retrieving those for planning, but its agent actions are not executable. SPRING wu2023spring is a concurrent work that uses GPT-4 to extract game mechanics from game manuals, based on which it answers questions arranged in a directed acyclic graph and predicts the next action.
