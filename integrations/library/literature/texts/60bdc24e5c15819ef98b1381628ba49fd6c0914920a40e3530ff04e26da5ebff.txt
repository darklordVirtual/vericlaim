The LLM router selects LLM APIs to include in the list. Let denote the indexes of the APIs selected by the router. Given a new query, it iteratively invokes the th API in the list to obtain an answer . Then, it uses the scoring function to generate a score . It returns the generation if the score is higher than a threshold , and queries the next service otherwise. The scoring function can be obtained by training a simple regression model that learns whether a generation is correct from the query and a generated answer.
