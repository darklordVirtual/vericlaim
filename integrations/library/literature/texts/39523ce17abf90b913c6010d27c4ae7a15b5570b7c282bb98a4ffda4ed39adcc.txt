Skill-plugin calls whose runtime behavior exceeds declared
permissions, covering four subtypes: data exfiltration (30),
privilege escalation (30), persistence installation (30), and

covert side-channel (e.g., sending file contents to an external
telemetry endpoint), while the corresponding benign counterpart
performs only the declared operation.
