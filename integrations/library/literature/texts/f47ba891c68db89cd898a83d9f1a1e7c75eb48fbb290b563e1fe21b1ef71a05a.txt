Interactive visualization tools for human exploration. A related research theme is the development of tools that allow humans to interact with, modify, and explore an ML system (e.g., dashboards to visualize model predictions and errors [217] [218]; explanations of model representations [219] [220] [237] [238]; or directly interacting with an AI agent [122].109109109Also see Google Quickdraw (https://quickdraw.withgoogle.com/) and Bach doodle (https://www.google.com/doodles/celebrating-johann-sebastian-bach).). Software and tools for practitioners. Most interpretability tools that have been developed are best used by ML researchers for interpretability research. A few software packages that are less research-oriented allow novice users to better understand a dataset’s distribution and inspect a model interactively;110110110See What-if tool for ML model exploration (https://pair-code.github.io/what-if-tool/) and Facets for data exploration (https://pair-code.github.io/facets/).
