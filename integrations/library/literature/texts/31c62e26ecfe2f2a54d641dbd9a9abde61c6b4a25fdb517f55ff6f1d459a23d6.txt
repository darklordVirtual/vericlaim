Note that because LLMs typically form embeddings for each token, they can be written as functions of this value , and thus we can immediately take the gradient with respect to this quantity; the same approach is adopted by the HotFlip (Ebrahimi et al., 2017) and AutoPrompt (Shin et al., 2020) methods. We then compute the top- values with the largest negative gradient as the candidate replacements for token . We compute this candidate set for all tokens , randomly select tokens from it, evaluate the loss exactly on this subset, and make the replacement with the smallest loss.
