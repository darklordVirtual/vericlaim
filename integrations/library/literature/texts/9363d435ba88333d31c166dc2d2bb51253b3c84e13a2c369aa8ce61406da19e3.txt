Many ML models we consider directly compute class probabilities as a continuous function of the input and real-valued model parameters. In this case, an API that reveals these class probabilities provides an adversary with samples that can be viewed as equations in the unknown model parameters. For a large class of models, these equation systems can be efficiently solved, thus recovering (or some good approximation of it). Our approach for evaluating attacks will primarily be experimental.
