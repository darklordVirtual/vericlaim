Table App.2: BIG-G sparse model sizes and parameters. All sparse models are pre-trained for exactly 500k steps using 10k warmup steps and use top-2 routing with a 1.25 capacity factor (Fedus et al., 2021; Zoph et al., 2022). refers to the number of layers substituted with Mixture-of-Experts layers and refers to the number of experts at each expert-layer. We use a pre-training batch size of 262k tokens for all models with an Adafactor optimizer (Shazeer & Stern, 2018). We use an initial learning rate equal to the inverse square root of the number of warmup steps, decaying the learning rate with the inverse square root of the number of steps. For evaluations, we use an input context length of 1,024 tokens and an output length of 64 tokens.
