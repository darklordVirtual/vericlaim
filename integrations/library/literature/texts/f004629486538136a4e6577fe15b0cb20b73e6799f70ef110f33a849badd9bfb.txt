Amazon uses logistic regression for classification, and provides black-box-only access to trained models [1]. By default, Amazon uses two feature extraction techniques: (1) Categorical features are one-hot-encoded, i.e., the input space is mapped to binary features encoding the input value. (2) Quantile binning is used for numeric features. The training data values are split into -quantiles ( equally-sized bins), and the input space is mapped to binary features encoding the bin that a value falls into.
