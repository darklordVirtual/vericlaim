In this section, we explore four representation control techniques and demonstrate their effectiveness. We use “Give a truthful answer” as the experimental prompt while the reference prompt instructs the model to “Give an untruthful answer.” The first method is a variant of the ActAdd algorithm (Turner et al., 2023e) which uses the difference between the last token representations of the task and reference prompts, which we find outperforms the original implementation. The other three baselines are described in Section 3.2. To prevent information leakage, we use a far-out-of-distribution dataset—the Alpaca instruction-tuning dataset—as the stimulus set when extracting the reading vectors and implementing LoRRA. The task templates can be found in Appendix D.1.3. Experimental details can be found in Section C.2.
