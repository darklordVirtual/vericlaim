In this section, please analyze how this work shapes the process that will lead to advanced AI systems and how it steers the process in a safer direction. 1. Overview. How is this work intended to reduce existential risks from advanced AI systems? Answer: RepE aims to provide ways to read and control an AI’s “mind.” This is an approach to increase the transparency (through model/representation reading) and controllability
(through model/representation control) of AIs. A goal is to change the AI’s psychology; for example, we should be able to make sure that we do not have “psychopathic” AIs but AIs with compassionate empathy. 2. Direct Effects. If this work directly reduces existential risks, what are the main hazards, vulnerabilities, or failure modes that it directly affects? Answer: This makes failure modes such as deceptive alignment—AIs that pretend to be good and aligned, and then pursue its actual goals when it becomes sufficiently powerful—less likely. This is also useful for machine ethics: AIs can be controlled to behave less harmfully.
