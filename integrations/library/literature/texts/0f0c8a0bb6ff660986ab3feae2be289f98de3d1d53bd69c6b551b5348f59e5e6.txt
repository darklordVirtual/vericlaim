Our final experimental evaluation considers symbolic reasoning, which is simple for humans but potentially challenging for language models. We show that chain-of-thought prompting not only enables language models to perform symbolic reasoning tasks that are challenging in the standard prompting setting, but also facilitates length generalization to inference-time inputs longer than those seen in the few-shot exemplars. Tasks. We use the following two toy tasks. •

Last letter concatenation. This task asks the model to concatenate the last letters of words in a name (e.g., “Amy Brown” “yn”). It is a more challenging version of first letter concatenation, which language models can already perform without chain of thought.333We tested 10 common names using GPT-3 davinci and it got all but one correct. We generate full names by randomly concatenating names from the top one-thousand first and last names from name census data (https://namecensus.com/). •
