this network. For comparison, a network of this size gets to errors
when regularized by weight decay alone and can be improved to around
by using carefully applied dropout. A subtle, but essential
detail is that we only got improvements by generating adversarial examples for
each layer outputs which were used to train all the layers above. The network was trained in an alternating fashion, maintaining and updating
a pool of adversarial examples for each layer separately in addition to the
original training set.
