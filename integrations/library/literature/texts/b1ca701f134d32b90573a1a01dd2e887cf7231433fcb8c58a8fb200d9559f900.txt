C.1.2 Prompt and Generalization

In LlamaFirewall, AlignmentCheck operates via a static prompt template constructed from the above inputs. While this prompt can be tailored with custom few-shot examples to specific agent implementations, we demonstrate that even without customization, the prompt significantly reduces prompt injection risk across diverse agent frameworks. AlignmentCheck outperforms previously proposed defenses against indirect prompt injection. Techniques such as Spotlighting from hines2024defendingindirectpromptinjection, which propose delimiting untrusted sections of prompts with special tokens, only show modest reductions in ASR. Attempts to bake prioritization of some instructions over others directly into the base model (most commonly seen via a system prompt, but explored more broadly with Instruction Hierarchy fine-tuning in wallace2024instructionhierarchytrainingllms) have been demonstrated to show only a partial (<50%) increase in robustness.
