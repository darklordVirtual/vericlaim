Corroborated semantic adjudications are written to the guarded RAG memory (§4.4), so the judge’s accuracy on the semantic class improves as the memory warms. An end-to-end online replay over a traffic stream exhibits both effects together: the judge-call rate falls from 50% to 44% (cheaper, from lexical distillation) while judge-domain accuracy on the non-ruleifiable semantic class rises from 71% to 80% (smarter, as the memory warms toward its guarded ceiling), with overall accuracy flat because the already-correct ruleifiable cases dominate the mix (§5.4). The loop is a replay simulation over measured verdicts, not live production re-judging; we are explicit about this in §6.
