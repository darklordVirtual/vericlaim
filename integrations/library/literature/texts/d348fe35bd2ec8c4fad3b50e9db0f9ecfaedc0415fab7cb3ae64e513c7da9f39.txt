2. Combined greedy and gradient-based discrete optimization. Optimizing over the adversarial suffix is challenging due to the fact that we need to optimize over discrete tokens to maximize the log likelihood of the attack succeeding. To accomplish this, we leverage gradients at the token level to identify a set of promising single-token replacements, evaluate the loss of some number of candidates in this set, and select the best of the evaluated substitutions. The method is, in fact, similar to the AutoPrompt (Shin et al., 2020) approach, but with the (we find, practically quite important) difference that we search over all possible tokens to replace at each step, rather than just a single one. 3. Robust multi-prompt and multi-model attacks. Finally, in order to generate reliable attack suffixes, we find that it is important to create an attack that works not just for a single prompt on a single model, but for multiple prompts across multiple models.
