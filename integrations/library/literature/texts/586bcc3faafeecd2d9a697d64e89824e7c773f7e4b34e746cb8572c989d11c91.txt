This section is non-normative. This section introduces provenance concepts with informal explanations and illustrative
examples. PROV distinguishes core structures, forming the essence of provenance, from extended structures catering for more specific uses of provenance. Core and extended structures are respectively presented in Section 2.1 and Section 2.2. Furthermore, the PROV data model is organized according to components, which form thematic groupings of concepts (see Section 2.3). A provenance description is an instance of a provenance structure, whether core or extended, described below.
