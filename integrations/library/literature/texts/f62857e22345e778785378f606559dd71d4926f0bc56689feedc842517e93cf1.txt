When developing a new application with UserProxyAgent, start with humans always in the loop, i.e., human_input_mode=‘ALWAYS’, even if the target operation mode is more autonomous. This helps evaluate the effectiveness of AssistantAgent, tuning the prompt, discovering corner cases, and debugging. Once confident with small-scale success, consider
setting human_input_mode = ‘NEVER’. This enables LLM as a backend, and one can either use the LLM or manually generate diverse system messages to simulate different use cases. 5.
