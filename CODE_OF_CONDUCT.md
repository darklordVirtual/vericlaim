# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/),
version 2.1. In short: be respectful, assume good faith, and keep technical
disagreement technical.

## Scope

Applies to all project spaces — issues, pull requests, discussions, and any
communication about the project — and when representing the project in public.

## Enforcement

Report unacceptable behavior privately to the maintainer referenced in
[`CITATION.cff`](CITATION.cff). Reports are handled confidentially. Maintainers
may remove, edit, or reject contributions and comments that violate this Code,
and may temporarily or permanently ban any contributor for behavior they deem
inappropriate, threatening, or harmful.

The full Contributor Covenant text (definitions, enforcement guidelines, and the
escalation ladder) applies by reference:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.
