# Bibliography

The works vericlaim's method and documentation actually lean on. Entries
carry no hand-typed DOIs or arXiv ids — identifiers are resolved and
verified against the registrar (Crossref/arXiv) by the curation pipeline,
with a strict title-agreement guard; an entry that cannot be verified stays
uncited. See `docs/manifesto.md` and `docs/design-notes/contract-lineage.md`
for where each idea is used.
